NYPL Open Hours Application

STEP 1: Choose one of the branches



STEP 2: Choose one of the two options:

A. Enter the number of days. The application will show if the branch is open after the number of days in the future.


B. Enter the date and time you want to chenck if the branch is open. Please follow the format:"YYYY-MM-DD HH:MM:SS" and the hour is military time format.


STEP 3: Enter the number of days or the time stamp



To get the URL endpoint of application along with the location slug, please select a branch above.

By entering the number of the days, you can see if the branch is open that number of days in the future from now. Default is 0. The input is not a number, not an integer or less than 0 will be set as default.

After done your selection, press "Confirm Selection" to get the URL.
